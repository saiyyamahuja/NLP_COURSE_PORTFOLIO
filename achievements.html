Projects

NewsGuardian: An Intelligent System for Fake News Detection

Developed an intelligent system that automatically identifies and classifies news articles as real or fake using NLP and machine learning techniques. The system processes text through tokenization, stopword removal, and lemmatization, followed by feature extraction using TF-IDF and word embeddings. Multiple classification models were trained and evaluated, with neural networks providing the best performance. A web-based interface built with Django and React allows users to check news content and includes an educational quiz module to promote media literacy.
