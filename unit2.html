Unit 2: Syntax Analysis

9 Hours

Unit Overview

This unit focuses on syntax analysis, which involves understanding the grammatical structure of sentences and how words combine to form meaningful phrases and sentences. Students learn about formal grammars, parsing techniques, and computational approaches to syntactic analysis.

Download Unit 2 Notes (PDF)

Context Free Grammars

Context-Free Grammars (CFGs) are formal systems for describing the syntactic structure of languages. This section covers:

  • Formal definition of CFGs (terminals, non-terminals, production rules, start symbol)
  • Derivations and parse trees
  • Chomsky Normal Form
  • Limitations of CFGs

Grammar Rules for English

This section explores how CFGs can be used to model English syntax:

  • Phrase structure rules (NP, VP, PP, etc.)
  • Handling subject-verb agreement
  • Dealing with complex constructions
  • Grammar development and refinement

Top-Down Parsing

Top-down parsing starts with the start symbol and attempts to derive the input string. This section covers:

  • Recursive descent parsing
  • Predictive parsing
  • LL parsing algorithms
  • Advantages and limitations of top-down approaches

Bottom-Up Parsing

Bottom-up parsing starts with the input string and attempts to reduce it to the start symbol. This section covers:

  • Shift-reduce parsing
  • LR parsing algorithms
  • Operator precedence parsing
  • Advantages and limitations of bottom-up approaches

Ambiguity

Ambiguity occurs when a sentence has multiple valid parse trees. This section covers:

  • Types of ambiguity (lexical, syntactic, semantic)
  • Identifying and resolving ambiguity
  • Impact of ambiguity on parsing
  • Techniques for handling ambiguous grammars

CKY Parsing

The Cocke-Kasami-Younger (CKY) algorithm is a dynamic programming approach to parsing. This section covers:

  • CKY algorithm formulation
  • Converting grammars to Chomsky Normal Form
  • Implementation and complexity analysis
  • Handling ambiguity in CKY parsing

Dependency Parsing

Dependency parsing focuses on the relationships between words rather than phrase structure. This section covers:

  • Dependency grammar formalism
  • Transition-based dependency parsing
  • Graph-based dependency parsing
  • Neural dependency parsing approaches
  • Universal Dependencies framework

Earley Parsing

Earley parsing is a dynamic programming algorithm that can handle any context-free grammar. This section covers:

  • Earley algorithm formulation
  • Chart parsing concepts
  • Implementation and complexity analysis
  • Advantages over other parsing approaches

Probabilistic Context-Free Grammars

Probabilistic Context-Free Grammars (PCFGs) extend CFGs by assigning probabilities to production rules. This section covers:

  • Formal definition of PCFGs
  • Parameter estimation from treebanks
  • Probabilistic parsing algorithms
  • Disambiguation using probabilities
  • Evaluation of probabilistic parsers
